## Getting Started

1. Clone this repository

```bash
git clone https://github.com/silentwraith03/portfolio.git
```

2. Navigate to the directory, then run

```bash
npm install
```

3. Run the Next.js development server:

```bash
cd portfolio
npm run dev
```
